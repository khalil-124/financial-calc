# Financial Calc Native 🏦

A professional-grade personal financial advisor and loan calculator mobile application built with **Kotlin** and **Jetpack Compose**.

## Features 🚀

- **Advanced Loan Engine**: Support for Reducing Interest, Flat Rates, Murabaha, and Rule of 78.
- **Insurance & Fees Integration**: Comprehensive modeling of upfront insurance, monthly premiums, annual taxes, and mandatory card fees.
- **Dynamic Dashboard**: Interactive 3-segment pie chart (Principal vs. Interest vs. Fees) for immediate cost visualization.
- **Smart Repayments**: Calculate the impact of balloon payments and monthly extra payments on loan term and interest savings.
- **Professional Reporting**: Generate and export detailed Monthly or Yearly amortization schedules as high-quality PDF files.
- **Fintech UI**: A premium, minimalist design system with Light and Dark mode support.

## Project Structure 🏗️

- `app/src/main/java/com/khalil/calc/logic`: Core financial computation engines and data models.
- `app/src/main/java/com/khalil/calc/ui`: Modern Jetpack Compose UI components and layouts.
- `app/src/main/java/com/khalil/calc/pdf`: PDF generation utilities and reporting templates.

## Tech Stack 🛠️

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Design Pattern**: MVVM (Model-View-ViewModel)
- **Tooling**: Gradle, Material 3, Android XML printing
